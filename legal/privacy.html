8. Меры защиты

  • SSL/TLS-шифрование (Cloudflare + Apache) — весь трафик передаётся по HTTPS.
  • Аутентификация по API-ключам для доступа к учебной платформе.
  • Row Level Security (RLS) в PostgreSQL — изоляция данных между пользователями.
  • Файрвол и ограничение доступа к серверу по SSH (нестандартный порт, ключи).
  • Логирование доступа к данным.
