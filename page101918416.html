    Найдешь более 15 багов в реальном API

    Документация — это ожидание. Ответы API — реальность. Твоя задача — найти, где они не совпадают. Ты получишь реальный навык тестирования и узнаешь слабые места 90% всех API.

    Ты научишься документировать найденные баги, используя разработанный шаблон. А результат отправишь на проверку и получишь подробную обратную связь.
