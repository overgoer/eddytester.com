### 4. [IDEA] Квест-уроки «Охота за багами без подсказки»

**Статус:** 💡 Идея. Не подтверждено. Требует ресерча.

**Концепция:** Уроки без видео — только текст + система проверки знаний. Студент получает текст:
«В API добавили новый эндпоинт POST /users/bulk. Документация — ссылка. Postman — коллекция. Найди баги сам.»
Система проверки (уже реализована в page101922726.html) подтверждает найденные баги.

**Что нужно для реализации:**
1. Добавить квестовые уроки в LESSON_MAP на странице курса (без iframe, только блок с текстом + тест)
2. В меню слева — иконка 🔍, бейдж «самостоятельно», нет video preview
3. В API — новые баги на квестовых эндпоинтах (если их нет)
4. В docs/bugs.html — добавить баги для квестовых эндпоинтов
5. После прохождения — бейдж «🎖 Самостоятельно» и/или достижение
6. Система мягких подсказок при повторных попытках (опционально)

**Бэкенд-затраты:**
- Минимальные — вся логика проверки уже есть (/save-progress, /get-progress, LESSON_MAP)
- Нужно только добавить конфиги для новых уроков в базу данных (или файл конфигурации)
- Если багов для квестовых эндпоинтов нет — внести их в v1 API

**Риски:**
- Студент может испугаться отсутствия видео
- Нужен A/B тест на 1-2 уроках перед полноценным запуском
